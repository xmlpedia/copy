Hardwarewallet

Trezor is the oldest and most reputable hardware wallet. Vulnerabilities are quickly fixed with updates because the code is open source so many people can look for bugs in the code.

There are two very important security measures that can be implemented with hardware wallets. 1. pass phrase 2. multisig

ABCore Full Node

How to run a Bitcoin full node on a cheap Android phone

All you need is a cheap Android phone with external SD Card capability like the Nokia 2.1 or 5.1 or the Motorola Moto G7 Play and a 400gb SD Card.

Step 1: Install ABCore from FDroid, Google Play store or https://github.com/greenaddress/abcore.

Install the File Manager app from Fdroid (or any other file manager app that can tell you the path of your external sd card)

Step 2: Open the ABCore app and download the Bitcoin Deamon.

Step 3: In Settings click on

If your phone goes into sleep mode and ABCore shuts down you can use an app like Wakelock v3 that keeps your cpu and wifi from shutting off in sleep mode.
